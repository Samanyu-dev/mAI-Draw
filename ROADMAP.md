# Roadmap

This roadmap describes the near-term work needed to make mAI-Draw easier to use, safer to contribute to, and more useful as an open source reference for AI-powered creative tools on iOS and iPadOS.

## 0.1.x - Public Foundation

- Improve README setup instructions
- Add screenshots and a short demo recording
- Document local secret configuration for Xcode
- Add issue and pull request templates
- Add secret scanning for pull requests
- Publish the first public release notes

## 0.2.x - Contributor Experience

- Add an architecture overview for the canvas, auth, sync, and AI services
- Document Supabase schema and local setup
- Add sample data for local testing
- Add CI validation for project files
- Add smoke-test instructions for iPad and iPhone targets

## 0.3.x - Product Hardening

- Improve privacy controls for audio and imported media
- Add clearer error states when AI credentials are missing
- Review token handling and auth flows
- Improve offline behavior and sync conflict handling
- Add regression tests for project import and export

## Longer-Term Ideas

- Plugin-like AI actions for canvas content
- Better markdown and structured note export
- Collaboration workflows
- Local-first storage mode
- Contributor examples for adding new AI providers
